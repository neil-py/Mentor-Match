
# Mentor Match

Prototype Verision 0.1 (developement on progress)

## Installation

Go to the app directory and use the package manager [pip](https://pip.pypa.io/en/stable/) to install the requirements.

```bash
pip install -r requirements.txt
```

## Usage

Run the application
```bash
python run.py
```
